# Takeaways

This folder is for reader contributions from the [Tool School: GitHub 101](https://hannahstulberg.substack.com/p/tool-school-github-101) article.

## How to Contribute

1. Fork this repo
2. Clone your fork
3. Create a branch
4. Create a new file: `[your-github-username].md`
5. Write one thing you learned, one takeaway, or how you're planning to use GitHub
6. Commit, push, and open a PR back to this repo

Your PR will show up on the real repo for everyone to see.
